Meu Primeiro Projeto da DIO - BOOTCAMP SANTANDER EXCEL COM IA 

Simulador de Investimentos em Fundos Imobiliários (FIIs)

Descrição do Projeto - 
Este repositório contém uma ferramenta de simulação financeira desenvolvida em Excel para análise de viabilidade e projeção de investimentos em Fundos Imobiliários. O objetivo principal é automatizar o cálculo de acumulação de patrimônio e geração de renda passiva com base em variáveis de mercado e perfil de investidor.

Objetivos Técnicos - 
Desenvolvimento de ferramentas de simulação de investimentos com foco em usabilidade.

Aplicação de matemática financeira para cálculo de rendimento mensal, dividendos e juros compostos.

Estruturação de dados para suporte à tomada de decisão estratégica.

Documentação técnica e versionamento de arquivos via GitHub.

Funcionalidades da Planilha - 
Cálculo de Projeção: Estimativa de patrimônio acumulado para períodos de 2, 5, 10, 20 e 30 anos.

Análise de Dividendos: Cálculo automático de retorno mensal esperado com base na taxa de rendimento informada.

Alocação de Ativos: Sugestão de distribuição percentual entre diferentes tipos de FIIs (Papel, Tijolo, Híbridos) conforme o perfil selecionado.

Dashboard de Acompanhamento: Visualização gráfica da evolução do montante ao longo do tempo.

Metodologia Aplicada - 
O projeto foi estruturado para resolver problemas comuns de investidores iniciantes, como a definição do valor necessário para investimento mensal e a compreensão do impacto do tempo no retorno financeiro. A solução utiliza fórmulas avançadas de Excel para processar entradas como salário, taxa de rendimento mensal e aporte sugerido.
